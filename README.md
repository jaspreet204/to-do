# Task Reminder
This project was created for Assignment 3 in Introduction to React SD-260.

## Project Description

Task Reminder is a simple to-do app made with React. Users can add, update, delete, and complete tasks. 
The app uses the useReducer hook to manage task state.

Tasks are also saved in localStorage, so they stay on the page after refreshing.

## Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Task date updates when edited
- Completed tasks show with different color
- Tasks save using localStorage
- Icons used for task actions

## Components

- AddTask
- TaskList
- TaskBox

## Technologies Used

- React
- JavaScript
- CSS
- useReducer
- localStorage
- react-icons
